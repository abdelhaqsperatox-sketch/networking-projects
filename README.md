# 🛠️ Practical Networking & Infrastructure Labs

Welcome! This repository serves as a practical portfolio of my hands-on experience with networking protocols, infrastructure configuration, and system administration workflows. 

I am actively building and documenting these labs to prepare for a **Fachinformatiker für Systemintegration** (IT Specialist for System Integration) apprenticeship in Germany. My focus is on understanding core network architectures, hardware configuration, and standard troubleshooting methodologies.

# 📂 Project Directory

Below is the structured overview of the infrastructure environments I have built, simulated, and validated:

 1. Small Office / Home Office (SOHO) Security & Access Control
* **Directory:** [`/SOHO-MAC-Filtering`](./SOHO-MAC-Filtering)
* **Core Technologies:** Cisco Packet Tracer, Layer 2 Switching, Wireless Security, Hardware Access Control (MAC Filtering).
* **Objective:** Simulating a standard small office network boundary, isolating endpoint components (Server, Printer, PC), and executing a rapid layer-2 mitigation technique against unauthorized peripheral access.

*🎨 More labs (Linux deployments, virtualization environments, services configuration) will be documented here as they are deployed.*

# ⚙️ Core Technical Focus Areas

* **Networking & Protocols:** OSI Model layers, TCP/IP suite, LAN/WAN segmentation, DHCP/NAT concepts, and basic switching/routing operations.
* **Simulations & Hardware:** Hands-on design using tools like Cisco Packet Tracer to mirror real-world corporate or small-business networking layouts.
* **Systems & Documentation:** Writing detailed, clear deployment notes—a vital skill for system documentation and compliance in German IT environments.
